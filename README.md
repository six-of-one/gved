G¹G²ved
=======

- visual editor for gauntlet, gauntlet 2 mazes
    (and eventually sanctuary...)

Some commands you could try:
* gved -i maze115
    (interactive system)
* gved -g2 -v maze1
    (view output.png)
* gved floor0
    (view output.png)
